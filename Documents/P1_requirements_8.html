Jon, Kristian, Silje and Vetle

20. september, 2021

Our client

Kristian og foodtrucken

Purpose and goals

Informere om og selge produkter fra foodtrucken

Audience

The website's typical user will be students attending events during "UKA". The students will use the website to find a map of the venues where UKAs events are beeing held and where the foodtruck is placed. They will also find the menu, place orders and find and overview of the opening hours. Since many of the events sell a lot of alcohol, the webpage should be simple to navigate for people who might be a bit intoxicated.

The content of the site and how it is organized

  1. "Navigation"
  2. "Home"
    The home page of the website
  3. "Menu"
  4. "About Us"
    This page will contain more information about the Foodtruck and the people running it. It will also contain some information about if and when the Foodtruck is recruiting for an upcoming UKA.
  5. "Contact Us"
    This page will contain the contact information of the people in charge of the Foodtruck, aswell as a form to make contact directly from the website. It will also again show the location of the Foodtruck, since for people navigating the site looking for where the Foodtruck is the Contact Us page would be a natural place to look if you didn't see it on the home page.

Functional Requirements

  • The website should show if or when the Foodtruck is open
  • A user should be able to place an order when the Foodtruck is open. With the order a user should be able to include a comment to the chef
  • The website should have a map that shows the location of the Foodtruck
  • The website should function equally on any type of device, be it a computer or phone
  • The website should tell users about who we are, what we do and about recruitment
  • A user should be able to contact us

Non-Functional Requirements

  • The website should have a navbar that visually shows where on the page you are
  • The menu should be organized and filterable
  • Every page needs to load in under a second
