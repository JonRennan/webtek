Changes

Change in information handling: We changed the information handling from URL to local storage, as we found the solution a better choice.

Change on menu.html: We removed the "Added to order" button to save some space and placed the "-" on the left side of the counter instead of the right side to make it easier to click on the right button, we also enlarged the buttons.

Change on order.html: The order summary is more similar to the menu page to reuse more code. The styling for the form is a bit different from planned, as we wanted to modernize it.

Change on contact_us.html: We changed the form to match the style on the form on order.html.

Design for mobile screen: We chose to change the design a bit for smaller screens to make it more readable.

Overall desing changes: We chose to change the font size, as the size we had planned for was too big.

Additional JavaScript file language.js We added a separate JavaScript file for language to make the code more readable.

Jon fulfilled his JavaScript requirementswith a function to make all menu items on menu.html. Silje made the main functions to change language and the base for the function to change one html file, and we all contributed to expand it to every page.
